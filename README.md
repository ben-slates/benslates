# 🖥️ Kali Linux Portfolio

A single-page interactive portfolio inspired by a Kali Linux desktop environment.  
Built using pure HTML, CSS, and JavaScript.

---

## 🚀 Features

- 🔐 Login screen simulation  
- 🖥️ Kali Linux–style desktop interface  
- 💻 Interactive terminal with custom commands  
- 📁 File manager system  
- 📝 Mousepad (text viewer)  
- 📄 Resume support (PDF / Base64)  
- 🎯 Clean hacker-style UI

## 📂 Tech Stack

- HTML5  
- CSS3  
- Vanilla JavaScript

## ⚙️ About the Project

This project is a **frontend simulation of a Kali Linux environment** designed to showcase:
- Personal information  
- Skills and tools  
- Cybersecurity interests  
- Interactive UI experience  

All functionality is implemented in the browser without any backend.
